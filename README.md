
## Описание задания

Целью работы является исследование качества автоматического распознавания речи (ASR) для русского языка с использованием модели **OpenAI Whisper**.  
Для этого были выбраны аудиофрагменты от нескольких спикеров, выполнено распознавание речи и рассчитаны стандартные метрики качества — **Word Error Rate (WER)** и **Character Error Rate (CER)**.

---

## Данные

- Язык: русский
- Количество спикеров: **5**
- Количество фрагментов на спикера: **2**
- Общее количество аудиофрагментов: **10**
- Длительность каждого фрагмента: **10–20 секунд**
- Формат аудио: `.mp3`
- Место выполнения эксперимента: **Google Colab**

Для каждого аудиофрагмента была выполнена:
1. Ручная транскрипция (прослушивание человеком)
2. Автоматическая транскрипция с помощью Whisper

---

## Используемая модель

- ASR-система: **OpenAI Whisper**
- Размер модели: **large**
- Язык распознавания задан явно: `language="ru"`

---

## Методология оценки

Для оценки качества распознавания использовались следующие метрики:

- **Word Error Rate (WER)**  
  Показывает долю ошибок на уровне слов (вставки, удаления, замены).

- **Character Error Rate (CER)**  
  Показывает долю ошибок на уровне символов.

Метрики вычислялись путём сравнения:
- `reference` — ручная транскрипция
- `whisper` — результат модели Whisper

## Анализ результатов

- Значение WER существенно выше CER, что является типичным для ASR-систем:
  модель часто ошибается на уровне слов, но сохраняет высокую точность на уровне символов.
- Наблюдается вариативность качества распознавания между спикерами, что может быть связано с:
  - индивидуальными особенностями речи;
  - темпом и интонацией;
  - качеством записи.
- В целом модель Whisper (large) демонстрирует стабильное качество распознавания русского языка на коротких фрагментах речи.
